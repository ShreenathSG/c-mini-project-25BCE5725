# Attendance Management System

## Description
A simple C program to manage student attendance. Teachers can add students, mark daily attendance, view attendance reports and identify students with low attendance or perfect attendance.

## Features
- Add new students with register number and name
- Mark attendance for all students (Present/Absent)
- View all students with attendance percentage
- List students with attendance below 75%
- Display class summary with average attendance
- Search for individual student details
- Show students with 100% attendance
- Save and load data from file

## How to Run
1. Compile the program: `gcc attendance.c -o attendance`
2. Run the program: `./attendance`
3. Follow the menu options to perform operations

## File Structure
- `attendance.c` - Main C source code
- `attendance.txt` - Data file (created automatically)
- `README.md` - This file

## Requirements
- GCC compiler or any C compiler
- Linux/Mac/Windows terminal
